About Forjit AI

A free AI tools platform built in India — for Indians, by an Indian developer.

What is Forjit AI?

Forjit AI (forjitai.in) is a free, browser-based AI platform with two core offerings:

🤖

AI Content Generator

Generate web apps, mobile apps, meal plans, workout routines, resumes, PDFs, PPTX slides from a text prompt.

🔧

41+ Free Tools

EMI calculator, GST calculator, SIP calculator, income tax, BMI, word counter, QR generator and more.

🇮🇳

Made for India

Indian tax slabs, GST, PPF, HRA, CGPA conversions — built with India-specific context.

🔒

100% Private

All calculations run in your browser. No data sent to servers. Your API key stays on your device.

The AI Generator

The AI generator on Forjit AI uses open-source large language models — Llama 3.3 70B, Qwen 3, DeepSeek V3 and others — via Groq and OpenRouter APIs.

Why "Forjit AI"?

The name Forjit comes from forging — like a blacksmith shapes metal, Forjit AI shapes raw prompts into useful outputs. The hammer logo (🔨) reflects this intent.

The platform is built and maintained by a solo developer in India, focused on making AI tools accessible — free, fast, and without unnecessary complexity.

Technology

Forjit AI is built with:

React + Vite Tailwind CSS Groq API OpenRouter API Llama 3.3 70B Qwen 3 DeepSeek V3 Deployed on Vercel

Privacy

Forjit AI respects your privacy:

✅ All calculator tools run entirely in your browser — no data is sent to any server.
✅ AI features use your own API key — Forjit AI never sees your key (stored in IndexedDB on your device only).
✅ No tracking beyond Google Analytics page views.
✅ No ads (yet — AdSense planned for the free tier).
✅ No mandatory login.
