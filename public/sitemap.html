Content Categories

🔨 Create App

Generate web & mobile apps from a text prompt. Full HTML output with inline CSS & JS.

📅 Content Planner

Meal plans (weekly/monthly), recipes with times & flame levels, workout schedules, diary apps. With localStorage persistence.

📄 Documents

Resumes, cover letters, reports, presentations, formal letters. Download as PDF, DOCX, or PPTX.

Features

  • Authentication — Mobile OTP-based login (email optional)
  • Community Gallery — Browse content generated by all users
  • User Uploads — Upload your own HTML apps (session-scoped)
  • Persistent History — IndexedDB-backed, survives refreshes
  • Export/Import — Backup all data as JSON
  • Admin Panel — User management, error logs, feedback review
  • PWA Support — Install as an app on Android & iOS
  • Error Logging — Track and export issues for debugging
  • Deploy to GitHub Gist — Host generated apps with live URL

Supported LLM Providers

  • Groq — Llama 3.3 70B, Llama 3.1 8B, Qwen 3 32B, DeepSeek R1
  • OpenRouter — Free tier models: Llama 3.3, Qwen 3 Coder, DeepSeek V3.1, Mistral Small
